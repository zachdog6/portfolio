Small Crud website to demostrate a react frontend, express backend, and a sqlite database.

Employee-details must be run on the same computer as the website. In reality this would be on a server, but as a demo this runs on the same computer.

npm required to run website, since it isn't hosted on a server.

installing npm: https://www.npmjs.com/get-npm

running:
	open employee_details folder
	open employee-details.exe to start express backend
	open cmd or console
	navigate to the employee_front folder
	run "npm install"
	run "npm start"
